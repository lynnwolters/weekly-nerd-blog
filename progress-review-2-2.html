May 8th, 2024

Three learning goals for the meesterproef formulated in SMART:

Goal 1: Improve planning skills

* Specific: I want to improve my ability to create effective plans to successfully finish projects
* Measurable: By the end of the master project, ensure we deliver what the client expects
* Achievable: It's a challenging but realistic goal
* Relevant: This skill is crucial for my future career
* Time-bound: Throughout the master project duration

How I Will Improve:

* Request leadership roles within my team
* Use the MoSCoW method to prioritize tasks
* Assign tasks in Trello based on team members' learning goals
* Set realistic deadlines for each team member
* Maintain deadlines through daily stand-ups and Trello updates

Goal 2: Get comfortable with JavaScript

* Specific: I aim to become more comfortable with writing JavaScript independently
* Measurable: By the end of the master project, write JavaScript as fluently as HTML/CSS without frequent lookup
* Achievable: Though it may impact the timeline, it's feasible with proper planning
* Relevant: Essential for creative coding and future job prospects
* Time-bound: Throughout the master project duration

How I Will Improve:

* Dedicate sufficient time and plan accordingly
* Avoid immediate lookup for solutions
* Break down tasks step-by-step and work through them
* Read comprehensive explanations when looking up information

Goal 3: Learn to implement accessibility

* Specific: I want to learn how to incorporate accessibility into projects
* Measurable: By the end of the master project, effectively apply HTML accessibility features and pass screen reader tests
* Achievable: Feasible if given the opportunity to implement in our project
* Relevant: Increasingly important in web development
* Time-bound: Throughout the master project duration

How I Will Improve:

* Request permission to implement accessibility features
* Study the MDN Web Docs for HTML attributes
* Conduct screen reader tests to ensure effectiveness
* These objectives outline your goals clearly and provide a structured approach to achieving them during your master project
